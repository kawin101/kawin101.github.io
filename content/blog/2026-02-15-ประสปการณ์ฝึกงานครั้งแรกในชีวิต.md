ผมเรียนอยู่สาขาวิทยาการคอมพิวเตอร์และสารสนเทศ ที่ มหาวิทยาขอนแก่น วิทยาเขตหนองคาย ตอนนั้น ผมอยู่ชั้นปีที่ 3 ตอนปิดเทอม ฤดูร้อน (เมษายน - มิถุนายน 2566) ทุกคนต้องหาที่ฝึกงาน เพราะเป็นวิชาบังคับ ก่อนจบ ที่ทุกคน ต้องมีประสบการณ์ทำงานจริงในสังคมแรงงานที่มีการแข่งขันดุเดือด \
\
ผมมีความโชคดีอย่างหนึ่ง ที่มีอาจารย์นพคุณและอาจารย์ศรัญญา แนะนำกลุ่ม Facebook ที่เป็นชุมชนที่พูดคุยกับ Software IT ธุรกิจ และการเปิดรับนักศึกษาฝึกงานจากทั่วประเทศไทย ที่มีเจ้าของกลุ่ม Facebook เป็น CEO บริษัทเครือ Botnoi Group ที่เป็นเจ้าของ Platform ที่ใช้ในการเรียน การสอน เกี่ยวกับการทำ LINE Chatbot อยู่ในขณะนั้น \
\
ผมโชคดี เด้งที่สอง คือ ผมได้ลองเขียน Resume เป็นภาษาอังกฤษ แล้วส่งใบสมัครผ่านอีเมลของบริษัท CEO เห็น Resume แล้วสนใจ โทรมาสัมภาษณ์สั้นๆ ประมาณ 5 นาที ตอนนั้น ผมตื่นเต้นมากๆ ผมพยายามตั้งสติ เพื่อตอบถาม คำถามก็ไม่ได้ยาก ถามว่าถนัดอะไร ชอบอะไร เคยทำโปรเจคอะไรมาบ้าง โชคดีที่ Skills และโปรเจคที่เคยทำ ตรงกับ TechStack ของบริษัท \
\
ต่อมา ผมผ่านสัมภาษณ์ ผมสมัครตำแหน่ง Developer แต่ได้เข้าฝึกงานตำแหน่ง UX/UI Designer แล้วรู้สึกงงๆ ว่าเกิดอะไรขึ้น แต่ก็ไม่เป็นไร ลองดูไม่เสียหาย เพื่อได้เรียนรู้อะไรใหม่ไปใช้ในอนาคต ตอนฝึกตำแหน่ง UX/UI ได้ออกแบบพวกหน้าต่างผู้ใช้ (UI) และประสบการณ์ผู้ใช้งาน UX ให้กับ LINE Chatbot และ Web App แล้วรู้สึกว่าไม่ใช้ อยากฝึก Developer อยากเขียนโค้ด \
\
พอดี ตอนนั้นในบริษัท มีกิจกรรมแข่งขันสร้าง LINE Chatbot ภายในองค์กร โชคดีมากๆ ถึงเวลาที่ผมจะได้โชว์ของแล้ว ว่าผมนะ เป็น Dev ได้นะ ผมได้จัดกลุ่มกับเพื่อนๆ  นักศึกษาฝึกงานคนอื่นๆ มีทั้งคนที่รับหน้าที่ออกแบบ คนเขียนโค้ด สร้างระบบ \
\
พอดีเลย ผมเรียนวิชา Software Engineering มาได้สร้าง LINE Chatbot มาแล้ว ผมเลยสามารถทำได้เลย พอจบกิจกรรม ก็จะได้เห็นผลงานของเพื่อนๆ ได้เห็นมุมมอง ความคิดของแต่ละทีม ว่าเป็นยังไง ได้รับคำแนะนำ แนวทางในปรับไปใช้งานจริง สำหรับงานลูกค้า จากพี่ๆ ภายในบริษัท \
\
พอจบกิจกรรม บริษัทกำลังหาคนที่สามารถเขียนโปรแกรมได้ เพื่อไปช่วยทีม Mobile Application Developer โชคดีเด้งที่ 3 ผมพึ่งเรียนวิชา Mobile App จบก่อนมาฝึกงาน ผมเลยพร้อมทำงานได้ทันที เพราะมีพื้นฐานอยู่แล้ว \
\
ก็แต่เจอปัญหาต่างๆ มากมาย ทั้งยาก ทั้งเหนื่อย แต่ก็ไม่เป็นไร ผมสนุกไปกับมันลุยๆ จอยๆ คิดว่าเป็นเรื่องท้าทายที่ต้องทำให้ได้ เพราะคนอื่นทำได้ เราก็ทำได้\
\
เวลาที่ผมเจออะไรที่ทำไม่เป็น ก็เรียนรู้หน้างานเลย ค้นหาใน Google บ้าง หรือ หาวิดีโอสอนใน YouTube บ้าง \
\
บ้างครั้งก็เจอ Error แปลกๆ ที่ถาม ChatGPT แล้ว มันตอบไม่ได้ ตอบมั่ว (ตอน 2019-2020) ผมเลยไปค้นหาใน StackOverflow ก็นั่งหาอยู่สักพักใหญ่ๆ 555 เจอละ คำตอบที่ผมหา ลองทำตามดู เอะ มัน work ทำงานได้ ดีใจสุดๆ เพราะครั้ง หรือ หลายครั้งๆ AI ยังตอบมั่วอยู่ เพราะมันไม่มีข้อมูล ผมเลยคิดว่ายังไงก็ต้องมีมนุษย์ทำงานสายโปรแกรมเมอร์อยู่ดี เพราะ AI มันเรียนรู้จากข้อมูล ส่วนมนุษย์เรียนรู้จากข้อผิดพลาดที่ไม่มีสิ้นสุด มนุษย์สามารถสร้างข้อมูลใหญ่ๆได้เรื่อยๆ ต่างจาก AI ที่ต้องเรียนข้อมูลจากมนุษย์ แต่ก็ไม่แน่ ไม่มีใครรู้อนาคต 100% อนาคต AI อาจจะมาช่วยเรา หรือทำลายเรา ก็ได้ ใครจะไปรู้ เพราะอนาคตไม่แน่นอน 555 \*\*ย้ำนะ ทุกคน ความคิดเห็นส่วนตัวล้วนๆ พอก่อนๆ นอกเรื่องมากไกลละ กลับเข้าเรื่องดีกว่า\
\
หลังฝึกงานผมได้ฝึกทั้ง UX/UI และ Mobile App ซึ่งเป็นประสบการณ์ที่คุ้มค่ามาก อยากขอบคุณ อาจารย์นพคุณและอาจารย์ศรัญญา ถ้าไม่มีอาจารย์ที่แนะนำผมวันนั้น คงไม่มีผมในวันนี้ ขอบคุณครับ
